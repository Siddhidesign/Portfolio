IRS2Go Redesign

Simplifying the American Tax Experience • Fall 2025

Context

Why IRS2Go?

Tax season is stressful. The IRS2Go app creates more anxiety than it solves due to fragmented navigation and unclear status updates.

1. System Analysis

I mapped the existing 5-step flow to identify where users were dropping off.

Discovery

Analyzing the "Big Two" Tasks

Task 1: Check Refund Status

We asked participants to simply "Check their status."

  • Users had to guess which tax year to select.
  • "Filing Status" used confusing legal terms.
  • No visual feedback after clicking "Get Status".

Task 2: Make a Payment

This was the highest friction point. Users were redirected to an external browser 3 times.

  • Users felt unsafe leaving the app to pay.
  • Too many payment processor options (Pay1040, ACI, etc.) without explanation.

Research Findings

Overall User Critique

The System Usability Score was low (3.5/5). The main violations were Error Prevention and User Control.

User Scenarios

I visualized "Alex's" journey to empathize with the first-time filer struggle.

The Solution

Redesigning Core Flows

1. Payment Flow

We moved the entire payment process inside the app. No more browser redirects. Users can now compare processor fees directly on one screen.

2. Refund Status

We cleaned up the form with clear labels and added a "Scan Document" feature to prevent typing errors.

Validation

Testing the Redesign

Methods & Data

We ran a second round of testing using Think-Aloud protocols. The results showed a massive spike in user satisfaction.

Feedback: Pros & Cons

While navigation improved, some users still missed the mandatory field indicators.

Final Iterations

Fixing the Last Friction Points

Iteration 1: Login vs. Guest

Insight: Users just want to check status, not sign up.

Fix: We made "Explore without signing in" the primary action.

Iteration 2: Payment Confirmation

Insight: "Did I just pay $500?" Users were scared to tap Pay.

Fix: Added a final Confirmation Modal to summarize fees before charging.
